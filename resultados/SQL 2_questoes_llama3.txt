**1. Pergunta:**  
Qual é o comando SQL utilizado para criar uma tabela no banco de dados?  

a) DELETE  
b) SELECT  
c) CREATE TABLE  
d) ALTER  

**Resposta correta:** c) CREATE TABLE  
**Explicação:** O comando `CREATE TABLE` é usado para definir a estrutura de uma tabela, incluindo colunas, tipos de dados e restrições, conforme exemplificado na seção "CREATE TABLE".  
---

**2. Pergunta:**  
Qual é o tipo de dado adequado para armazenar uma data no SQL?  

a) Char  
b) Float  
c) Date  
d) Text  

**Resposta correta:** c) Date  
**Explicação:** O tipo `DATE` é especificado para armazenar valores de data, como "2024-02-17", conforme visto na seção "CREATE TABLE" e "Tipos de Dados".  
---

**3. Pergunta:**  
Qual comando SQL é usado para inserir dados em uma tabela?  

a) SELECT  
b) UPDATE  
c) INSERT  
d) DELETE  

**Resposta correta:** c) INSERT  
**Explicação:** O comando `INSERT` permite adicionar novos registros a uma tabela, como mostrado no exemplo da seção "INSERT".

**4. Pergunta:**  
Qual é a função do comando `NOW()` em SQL?  

a) Excluir um registro  
b) Atualizar um campo  
c) Inserir uma data e hora atual  
d) Ordenar resultados  

**Resposta correta:** c) Inserir uma data e hora atual  
**Explicação:** O `NOW()` retorna a data e hora atual do sistema, utilizado no campo `registro` do exemplo da seção "INSERT".  
---

**5. Pergunta:**  
Qual é o propósito do uso do asterisco (*) no comando `SELECT`?  

a) Excluir colunas  
b) Selecionar todas as colunas  
c) Ordenar resultados  
d) Limitar o número de registros  

**Resposta correta:** b) Selecionar todas as colunas  
**Explicação:** O `*` no `SELECT` indica que todas as colunas da tabela serão retornadas, conforme exemplificado na seção "SELECT".
