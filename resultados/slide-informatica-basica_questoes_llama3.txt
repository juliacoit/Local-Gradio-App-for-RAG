**1. Pergunta:**  
O que é a CPU (Unidade Central de Processamento)?  

a) Um dispositivo de armazenamento de dados  
b) Um componente volátil que armazena temporariamente informações  
c) Conjunto de chips de silício que controla o fluxo de funcionamento da máquina  
d) Um periférico utilizado para entrada de dados  

**Resposta correta:** c) Conjunto de chips de silício que controla o fluxo de funcionamento da máquina  
**Explicação:** A CPU é responsável por coordenar e controlar as operações do computador, sendo o núcleo da unidade central.

**2. Pergunta:**  
Qual característica define corretamente a Memória RAM?  

a) É uma memória não volátil, onde os dados são armazenados permanentemente  
b) É uma memória volátil, permitindo leitura, gravação e regravação de dados  
c) Armazena dados de forma permanente, mesmo após desligar o computador  
d) É um dispositivo de armazenamento secundário, como discos rígidos  

**Resposta correta:** b) É uma memória volátil, permitindo leitura, gravação e regravação de dados  
**Explicação:** A RAM (Random Access Memory) é volátil, ou seja, perde seus dados ao desligar o computador.

**3. Pergunta:**  
Qual dispositivo é considerado o principal para armazenamento de informações em um computador?  

a) CD-ROM  
b) Disco Rígido  
c) Unidade de disco flexível  
d) Memória ROM  

**Resposta correta:** b) Disco Rígido  
**Explicação:** O disco rígido é o dispositivo principal de armazenamento de dados, capaz de guardar grandes quantidades de informações de forma permanente.

**4. Pergunta:**  
Quantos bits compõem um byte?  

a) 8 bits  
b) 10 bits  
c) 16 bits  
d) 32 bits  

**Resposta correta:** a) 8 bits  
**Explicação:** O byte é a unidade básica de armazenamento de dados, composto por 8 bits.

**5. Pergunta:**  
Qual exemplo de software pertence ao tipo "Sistema Operacional"?  

a) Microsoft Word  
b) Windows  
c) Adobe Photoshop  
d) Excel  

**Resposta correta:** b) Windows  
**Explicação:** O Sistema Operacional é o software responsável por gerenciar os recursos do computador e permitir a interação com o usuário. Windows é um exemplo clássico desse tipo.
