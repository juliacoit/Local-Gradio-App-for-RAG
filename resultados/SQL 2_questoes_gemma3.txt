**1. Pergunta:**  
Qual é a diferença principal entre os tipos de dados `CHAR` e `VARCHAR` em SQL?  

a) `CHAR` armazena texto fixo, enquanto `VARCHAR` armazena texto variável.  
b) `CHAR` é usado apenas para datas, enquanto `VARCHAR` é usado para números.  
c) `VARCHAR` é usado para armazenar imagens, enquanto `CHAR` é usado para armazenar som.  
d) `CHAR` é uma abreviação de `CHARACTER`, enquanto `VARCHAR` é uma abreviação de `VARIABLE CHARACTER`.  

**Resposta correta:** a)  
**Explicação:** `CHAR` armazena uma quantidade fixa de caracteres, enquanto `VARCHAR` permite que o tamanho da string varie.

**2. Pergunta:**  
Qual comando do DDL é utilizado para modificar a estrutura de uma tabela existente?  

a) `CREATE`  
b) `DROP`  
c) `ALTER`  
d) `SELECT`  

**Resposta correta:** c)  
**Explicação:** O comando `ALTER` é usado para modificar tabelas, como adicionar ou remover colunas.  

---

**3. Pergunta:**  
Qual comando do DML é usado para recuperar dados de uma tabela?  

a) `INSERT`  
b) `UPDATE`  
c) `SELECT`  
d) `DELETE`  

**Resposta correta:** c)  
**Explicação:** O comando `SELECT` é utilizado para consultar e recuperar dados armazenados em tabelas.

**4. Pergunta:**  
Qual comando é necessário para criar um banco de dados no SQL?  

a) `USE`  
b) `ALTER DATABASE`  
c) `CREATE DATABASE`  
d) `DROP DATABASE`  

**Resposta correta:** c)  
**Explicação:** O comando `CREATE DATABASE` é usado para criar um novo banco de dados.

**5. Pergunta:**  
Qual função é utilizada para registrar automaticamente a data e hora atual em um campo do tipo `TIMESTAMP`?  

a) `NOW()`  
b) `CURRENT_DATE()`  
c) `DATE()`  
d) `TIME()`  

**Resposta correta:** a)  
**Explicação:** A função `NOW()` retorna a data e hora atuais, ideal para campos `TIMESTAMP`.
