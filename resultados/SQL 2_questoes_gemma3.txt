Qual dos seguintes tipos de dados é utilizado para armazenar texto longo em SQL?  

a) Char

c) Text  
d) Int  

**Resposta correta:** c) Text  
**Explicação:** O tipo **Text** é usado para armazenar grandes volumes de texto, enquanto **Char** e **Varchar** são utilizados para strings com limites de caracteres definidos ou variáveis, respectivamente. O **Int** é para números inteiros.

**2. Pergunta:**  
Qual comando do DDL é utilizado para criar uma nova tabela no banco de dados?  

a) Select  
b) Drop  
c) Create  
d) Alter  

**Resposta correta:** c) Create  
**Explicação:** O comando **Create** pertence ao DDL (Data Definition Language) e é responsável por definir a estrutura de tabelas, índices e outros objetos no banco de dados.

**3. Pergunta:**  
Qual dos seguintes comandos é usado para recuperar dados de uma tabela em SQL?  

a) Insert  
b) Update  
c) Select  
d) Delete  

**Resposta correta:** c) Select  
**Explicação:** O comando **Select** faz parte do DML (Data Manipulation Language) e é utilizado para buscar e recuperar dados de uma tabela. Os outros comandos são usados para inserir, atualizar ou excluir registros.

**4. Pergunta:**  
Qual tipo de dado é adequado para armazenar datas e horas em SQL?  

a) Date  
b) Time  
c) Timestamp  
d) Boolean  

**Resposta correta:** c) Timestamp  
**Explicação:** O tipo **Timestamp** armazena tanto a data quanto a hora, enquanto **Date** só armazena datas e **Time** só armazena horários. O **Boolean** é usado para valores lógicos (verdadeiro/falso).

**5. Pergunta:**  
Qual comando do DDL é usado para excluir uma tabela existente?  

a) Alter  
b) Drop  
c) Create  
d) Select  

**Resposta correta:** b) Drop  
**Explicação:** O comando **Drop** pertence ao DDL e é utilizado para remover objetos do banco de dados, como tabelas, índices ou constraints. O **Alter** modifica a estrutura de objetos existentes.
