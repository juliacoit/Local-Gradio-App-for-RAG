**2. Pergunta:**  
Qual tipo de dado em SQL é adequado para armazenar datas?  

a) Char  
b) Date  
c) Float  
d) Text  

**Resposta correta:** b) Date  
**Explicação:** O tipo Date é especificado para armazenar valores de data, enquanto Char, Float e Text são usados para strings, números e textos longos, respectivamente.

**Resposta correta:** c) INSERT INTO

**4. Pergunta:**  
Qual comando é usado para recuperar dados de uma tabela?  

a) DELETE  
b) UPDATE
