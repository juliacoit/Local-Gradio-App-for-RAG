**1. Pergunta:**  
Qual dos seguintes tipos de dados é utilizado para armazenar uma string de texto com comprimento fixo?  

a) Varchar  
b) Text  
c) Char  
d) Float  

**Resposta correta:** c) Char  
**Explicação:** O tipo `Char` é usado para armazenar strings de texto com comprimento fixo, enquanto `Varchar` permite comprimento variável, `Text` é para textos longos e `Float` é para números reais.

**2. Pergunta:**  
Qual comando do DDL é utilizado para criar uma nova base de dados?  

a) ALTER DATABASE  
b) DROP DATABASE  
c) CREATE DATABASE  
d) USE DATABASE  

**Resposta correta:** c) CREATE DATABASE  
**Explicação:** O comando `CREATE DATABASE` é parte do DDL (Data Definition Language) e é usado para criar uma nova base de dados no SQL.  

---  
**3. Pergunta:**  
Qual comando do DML é utilizado para selecionar dados de uma tabela?  

a) DELETE  
b) UPDATE  
c) SELECT  
d) INSERT  

**Resposta correta:** c) SELECT  
**Explicação:** O comando `SELECT` pertence ao DML (Data Manipulation Language) e é usado para recuperar dados de uma ou mais tabelas.

**4. Pergunta:**  
Qual comando é utilizado para escolher uma base de dados para uso imediato?  

a) CREATE DATABASE  
b) DROP DATABASE  
c) ALTER DATABASE  
d) USE  

**Resposta correta:** d) USE  
**Explicação:** O comando `USE` é utilizado para selecionar uma base de dados já criada e torná-la a base de dados ativa para operações subsequentes.  

---  
**5. Pergunta:**  
Qual tipo de dado armazena tanto data quanto hora em um campo?  

a) Date  
b) Time  
c) Timestamp  
d) Char  

**Resposta correta:** c) Timestamp  
**Explicação:** O tipo `Timestamp` armazena data e hora simultaneamente, enquanto `Date` e `Time` são usados para armazenar apenas data ou hora, respectivamente.
